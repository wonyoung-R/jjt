# 구글 애드센스 배치 전략 📊

## 🎯 추천 광고 위치 (우선순위 순)

### 1. **메인 페이지 - 필터 패널과 시설 목록 사이** ⭐⭐⭐⭐⭐
**위치**: FilterPanel 아래, 시설 카드 그리드 위
```
[검색창]
[필터 패널]
━━━━━━━━━━━━━━━━━━━━━━
[광고 - 인피드 광고 (In-feed Ad)]
━━━━━━━━━━━━━━━━━━━━━━
[시설 카드 1] [시설 카드 2] [시설 카드 3]
```

**장점:**
- ✅ 사용자가 필터 선택 후 자연스럽게 시선이 가는 위치
- ✅ 콘텐츠와 자연스럽게 어우러짐
- ✅ 높은 가시성 (Above the fold)
- ✅ 클릭률(CTR) 높음

**광고 타입:** 
- 반응형 디스플레이 광고 (728x90 또는 300x250)
- 인피드 광고 (네이티브 광고)

**예상 수익:** ⭐⭐⭐⭐⭐

---

### 2. **시설 카드 그리드 중간 (3번째, 9번째, 15번째...)** ⭐⭐⭐⭐⭐
**위치**: 시설 카드 사이에 자연스럽게 삽입
```
[시설 1] [시설 2] [시설 3]
[시설 4] [시설 5] [광고 카드]
[시설 7] [시설 8] [시설 9]
```

**장점:**
- ✅ 가장 자연스러운 배치 (네이티브 광고)
- ✅ 스크롤하면서 자연스럽게 노출
- ✅ 시설 카드와 동일한 디자인으로 위화감 없음
- ✅ 모바일에서도 효과적

**광고 타입:**
- 인피드 광고 (추천!)
- 반응형 디스플레이 (카드 크기에 맞춤)

**예상 수익:** ⭐⭐⭐⭐⭐

---

### 3. **게시판 목록 - 게시글 사이** ⭐⭐⭐⭐
**위치**: 게시글 카드 사이 (2번째, 5번째, 8번째...)
```
[게시글 1 - 신메뉴 출시]
[게시글 2 - 할인 이벤트]

[광고 - 인피드 광고]

[게시글 3 - 모임 공지]
```

**장점:**
- ✅ 게시글 읽는 사용자의 관심도 높음
- ✅ 체류 시간이 긴 페이지
- ✅ 자연스러운 광고 노출

**광고 타입:**
- 인피드 광고
- 기사 내 광고 (Article Ad)

**예상 수익:** ⭐⭐⭐⭐

### 4. **게시판 상세 페이지 - 본문 하단** ⭐⭐⭐⭐

**장점:**
- ✅ 게시글을 끝까지 읽은 사용자 = 높은 관심도
- ✅ 다음 행동 전 자연스러운 광고 노출
- ✅ 클릭 가능성 높음

**광고 타입:**
- 디스플레이 광고 (300x250, 336x280)

## ⚠️ 주의사항

### 애드센스 정책 준수
1. **클릭 유도 금지**: "광고를 클릭해주세요" 등 금지
2. **광고 라벨**: "광고" 또는 "Sponsored" 표시 필수
3. **콘텐츠 정책**: 성인, 도박, 불법 콘텐츠 금지
4. **클릭 사기 금지**: 자가 클릭, 봇 클릭 금지

### 사용자 경험 우선
- 광고가 너무 많으면 사용자 이탈 증가
- 콘텐츠와 광고 비율: 70:30 유지
- 모바일에서 광고가 화면 50% 이상 차지 금지

### 성능 최적화
- 광고 lazy loading 적용
- 광고 스크립트 비동기 로드
- 페이지 로딩 속도 모니터링
