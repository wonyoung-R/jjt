# 🚀 The Lunch Captain (과천 지정타 점심/편의시설 가이드) 프로젝트 리포트

## 1. 프로젝트 개요
과천 지식정보타운(지정타) 직장인들의 최대 고민인 "오늘 점심 뭐 먹지?"를 해결하고, 나아가 지역 커뮤니티와 액티비티를 연결하는 로컬 플랫폼입니다.

## 2. 핵심 기능 (구현 완료)

### 🍱 맛집 큐레이션 & 검색
- **Google Sheets 기반 데이터 관리**: 별도의 백엔드 없이 스프레드시트로 데이터를 관리하여 운영 효율성 극대화.
- **상세 필터링**: 대분류(한식, 중식 등) > 중분류 > 소분류 필터 제공.
- **실시간 검색**: 장소명, 메뉴, 주소 등으로 즉시 검색 가능.

### 💖 My Picks (나만의 리스트)
- **로그인 없는 찜하기**: `localStorage`를 활용하여 회원가입 없이도 관심 매장을 저장.
- **직관적인 UI**: 카드 우측 상단의 하트 아이콘으로 간편하게 추가/삭제.
- **모아보기**: 필터 패널의 `My Picks` 탭에서 찜한 목록만 별도로 확인 가능.

### 🔗 Viral Loop (공유 기능)
- **팀장님 결재용 텍스트 복사**: `My Picks`에 담은 리스트를 깔끔한 텍스트로 변환하여 클립보드에 복사.
- **URL 포함**: 각 식당의 지도 링크(카카오맵/네이버)가 자동으로 포함되어 팀원 공유 시 편의성 증대.
- **문구**: `[오늘 점심 이거 어때? - The Lunch Captain]`

### 📢 사장님 & 유저 제보 시스템 (B2B)
- **제보 모달**: 메인 상단의 "사장님이신가요?" 배너를 통해 접근.
- **확장된 카테고리**: 
    - `맛집 홍보`: 사장님 직접 등록
    - `이벤트/할인`: 타임세일 등 정보
    - `모임/액티비티`: 원데이 클래스, 산책 모임 등 (커뮤니티 확장 대비)
- **Google Apps Script 연동**: 제보 내용은 즉시 Google Sheets로 전송되어 관리자가 확인 가능.

### 📬 이메일 구독 시스템 (Beta)
- **뉴스레터 구독**: 이메일 주소만으로 간편하게 구독 신청.
- **목적**: 회원가입 없이 잠재 고객(Lead) DB를 확보하여 추후 마케팅 및 알림 발송에 활용.
- **위치**: 모바일 리스트 중간 및 PC 화면 우측 하단(플로팅)에 배치 (현재 캐시 이슈로 노출 조정 중).

## 3. 마케팅 및 확장 전략 (Roadmap)

### Phase 1. 트래픽 확보 (Traffic First) - *Current Stage*
- **타겟**: 점심 메뉴 결정권자 (막내 사원, 팀장).
- **전략**: 
    - 오픈채팅방, 블라인드 등에 `My Picks` 텍스트 공유 기능을 활용하여 자연스럽게 바이럴.
    - "오늘 뭐 먹지?"라는 매일의 고통을 해결해주는 도구로 포지셔닝.

### Phase 2. 커뮤니티 & 액티비티 (Community Build)
- **타겟**: 지정타 내 취미 생활 및 네트워킹 니즈가 있는 직장인.
- **전략**:
    - 제보 시스템을 통해 수집된 '이벤트/모임' 정보를 게시판(`Board`)에 활발하게 노출.
    - 구독자 대상 뉴스레터 발송 (주간 핫플레이스, 모임 소개).

### Phase 3. 이동 & 연결 (Mobility & O2O)
- **타겟**: 교통 불편을 겪는 입주민 및 직장인.
- **전략**:
    - 축적된 트래픽과 유저 데이터를 기반으로 셔틀버스 수요 조사 및 매칭 서비스 도입.
    - 지역 상권(공방, 카페 등)과 연계한 쿠폰/예약 비즈니스 모델 도입.

## 4. 기술 스택
- **Frontend**: React, Tailwind CSS, Vite
- **Backend/DB**: Google Sheets API (via Google Apps Script)
- **Hosting**: GitHub Pages / Netlify (예정)
